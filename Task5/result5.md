# Задание 5. Описываем тесты

Набор интеграционных и end-to-end тестов для системы обработки платежей.

Сервисы:

- Payment Service      - управление счетами (списание, возврат) и переводы контрагентам
- FraudCheck Service   - проверка операций на мошенничество
- Notification Service - уведомления службы безопасности
- Saga Orchestrator    - координация Saga-процесса

## Набор тестов

| Название                                                      | Тип         | Компоненты                                                                   | Предусловия                                                                           |
| ------------------------------------------------------------- | ----------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| Успешный платеж от начала до конца                            | E2E         | Payment Service, FraudCheck Service, Saga Orchestrator                       | Счет клиента имеет достаточный баланс, операция проходит проверку на мошенничество    |
| Обнаружение мошенничества с компенсацией                      | E2E         | Payment Service, FraudCheck Service, Notification Service, Saga Orchestrator | Счет клиента имеет баланс, операция определена как мошенническая                      |
| Ошибка списания средств из-за недостаточного баланса          | E2E         | Payment Service, Saga Orchestrator                                           | Баланс счета клиента меньше суммы платежа                                             |
| Критическая ошибка перевода контрагенту                       | E2E         | Payment Service, FraudCheck Service, Saga Orchestrator                       | Средства списаны, проверка пройдена, перевод завершился необратимой ошибкой           |
| Повторная попытка проверки мошенничества при временной ошибке | Integration | FraudCheck Service, Saga Orchestrator                                        | FraudCheck Service возвращает временную ошибку, затем успешный ответ                  |
| Повторная попытка возврата средств при временной ошибке       | Integration | Payment Service, Saga Orchestrator                                           | Возврат средств не удался с первой попытки из-за временной ошибки                     |
| Повторная попытка уведомления при временной ошибке            | Integration | Notification Service, Saga Orchestrator                                      | Notification Service недоступен при первой попытке уведомления                        |
| Идемпотентность операций Payment Service                      | Integration | Payment Service, Saga Orchestrator                                           | Повторные запросы с одинаковым идентификатором транзакции                             |
| Timeout при взаимодействии с сервисами                        | Integration | Payment Service, FraudCheck Service, Saga Orchestrator                       | Один из сервисов не отвечает в течение установленного timeout                         |
| Максимальное количество повторов при временных ошибках        | Integration | FraudCheck Service, Notification Service, Saga Orchestrator                  | Сервис недоступен постоянно, достигнут лимит retry                                    |
| Корректность переходов State Machine                          | Integration | Saga Orchestrator                                                            | Проверка всех переходов состояний согласно спецификации                               |
| Восстановление после сбоя Saga Orchestrator                   | E2E         | Saga Orchestrator, все сервисы                                               | Оркестратор упал на промежуточном этапе, после перезапуска продолжает с того же места |
