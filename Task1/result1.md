# Задание 1. Проектируем шаги Saga

Таблица-реестр транзакций Saga-оркестрации.

| Транзакция               | Описание                                                                         | Тип            | Компенсация     |
| ------------------------ | -------------------------------------------------------------------------------- | -------------- | --------------- |
| DEBIT_CUSTOMER           | Списание денег со счета пользователя при инициации платежа                       | Компенсируемая | REFUND_CUSTOMER |
| REFUND_CUSTOMER          | Возврат денег пользователю при блокировке подозрительной операции                | Повторяемая    | -               |
| FRAUD_CHECK              | Проверка операции в антифрод-системе и внешних сервисах на предмет мошенничества | Повторяемая    | -               |
| NOTIFY_SECURITY          | Уведомление службы безопасности о заблокированной подозрительной операции        | Повторяемая    | -               |
| TRANSFER_TO_COUNTERPARTY | Перевод денег на контрагенту после успешной проверки                             | Необратимая    | -               |
